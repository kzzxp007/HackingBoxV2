# 🔥 HackingBox V5

HackingBox é uma ferramenta CLI (Command Line Interface) feita em Python que reúne diversos utilitários e recursos de segurança ofensiva e pentest em um único lugar, com menus interativos e simples de usar.

> ⚠️ Projeto educacional. Use apenas em ambientes controlados e com autorização.

---

## 📌 Versão
- **Nome:** HackingBox
- **Versão:** V5
- **Linguagem:** Python 3
- **Plataforma:** Termux / Linux
- **Interface:** CLI (Terminal)
- **Licença:** MIT

---

## 🚀 Funcionalidades

### 🕵️ OSINT
- Whois
- theHarvester
- Sherlock
- Amass

### 🌐 NMAP
- Instalação rápida do Nmap para análise de rede

### 💥 DDoS (educacional)
- NAMI-DDoS
- DDoSlayer
- HULK
- 60 ferramentas listadas

### 🦠 Malware (referência)
- Metasploit Framework
- Repositórios de malware (somente estudo)

### 🔓 Exploração de Vulnerabilidades
- SQLMap
- Nuclei
- Nikto
- XSStrike
- WPScan
- Subfinder

### 🎭 Phishing (educacional)
- ZPhisher
- MaskPhish
- Cloudflared

### 🎮 Game embutido
- Jogo de adivinhação de números (1 a 10)

### ⚙️ Sistema
- fastfetch
- Tor

---

## 🧠 Como usar

bash
git clone https://github.com/kzzxp007/HackingBoxV2
cd hackingbox
pip install pyfiglet colorama
python hackingbox.py

# 📟 Comandos internos
help → menu principal
tabelas → menu de ferramentas
game → inicia jogo
about → informações do sistema
system → fastfetch
tor → inicia Tor
exit → sair
